# TDRホテル宿泊／レストラン予約 攻略情報

## 概要

TDRの希少価値が高く予約確保が困難なホテル宿泊とレストラン予約に役立つ情報を記載します。
具体的に言うと、宿泊ならミラコスタのテラスルーム・ハーバーグランドビュー／ハーバービュー、バルコニールーム・ハーバービュー、レストランならハイピリオン・ラウンジやドリーマーズ・ラウンジ辺りの予約確保を目標としている方に向けた情報です。

このサイトを見て何かを実践すれば必ずしも確実に予約が取れるということはありませんが、最低限度の知識や対策を把握することで争奪戦のスタートラインに立てる程度にはなれると考えております。
むしろ、このサイトの一番の存在意義は色々と情報に触れることで『自分も拾えるのではないか？』というモチベーションの維持、つまり**心が折れそうなときに自分を奮起させること**なのかも知れません。

## マシュマロにお返事

マシュマロを頂けたようで、ありがたいことですね。不定期に古い書き込みは消していきますが、悪しからずご了承ください。

* 宿泊予約における同日の複数保持ですね、意外なことに公式のヘルプやよくある質問にピンポイントな情報が見つかりませんでした
    * 結論から言うと、同日の宿泊予約は複数確保しておくことが可能です
    * 例えば[予約サイトのヘルプの客室と人数制限について](https://reserve.tokyodisneyresort.jp/help/hotel_02#limit)によると、『1回の予約で、3部屋、5泊までの予約が可能』とありますが、これは1回の予約操作で取得可能な同日程／同種部屋の制限ですね
    * 小さい文字の注意事項には『4部屋、6泊以上の予約をご希望の場合には、複数回に分けての予約手続きが必要』とあるので、複数に分けての予約手続きで同日程の複数予約が可能である、と読み取れるかと思います
        * ちなみに1予約で複数部屋を掴むと部屋代表者の氏名入力が必要で同じ名前を設定できなかったとうろ覚え、別予約の場合ならそういった制限がなかった気がします
        * ちなみに『何枠まで持っておけるか』という上限枠数の情報も公式サイトだと見つけられませんでした
    * ただ、日程が固定で先に部屋を確保して、後から別の部屋に変更するなら素直にトラベルバッグ等から『変更』を手続きする手もあります
    * 別の予約で取ってしまうと元々の宿泊予約の『特典として予約したレストラン予約』がそのままでは引き継がれませんが、予約した部屋の変更ならこれを維持したまま変更可能なはずです
        * 部屋Aの予約特典で確保したレストランを部屋Bに移して欲しい、と問い合わせ窓口にTELで依頼をすれば対応してもらえた、という話もあるとかないとか…
    * ちなみにレストラン予約は同日の同時間帯（朝昼夕）を重複して確保は出来ませんね（グループ化した同行者の別アカウントで予約してもらうことは可能）
