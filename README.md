# TDRホテル宿泊／レストラン予約 攻略情報

## 概要

TDRの希少価値が高く予約確保が困難なホテル宿泊とレストラン予約に役立つ情報を記載します。
具体的に言うと、宿泊ならミラコスタのテラスルーム・ハーバーグランドビュー／ハーバービュー、バルコニールーム・ハーバービュー、レストランならハイピリオン・ラウンジやドリーマーズ・ラウンジ辺りの予約確保を目標としている方に向けた情報です。

このサイトを見て何かを実践すれば必ずしも確実に予約が取れるということはありませんが、最低限度の知識や対策を把握することで争奪戦のスタートラインに立てる程度にはなれると考えております。
むしろ、このサイトの一番の存在意義は色々と情報に触れることで『自分も拾えるのではないか？』というモチベーションの維持、つまり**心が折れそうなときに自分を奮起させること**なのかも知れません。

## マシュマロにお返事

マシュマロを頂けたようで、ありがたいことですね。不定期に古い書き込みは消していきますが、悪しからずご了承ください。

* ありがとうございます、スプホが落ち着いたので最近は新しい対策等も書いていないですし、面白ネタもたまに書かないとページの存在意義が…（苦笑
* 確かに『ちゃんと調べましょうね』なのですよね…（苦笑
    * 同情はしますが、公式のページもまとまりが悪いというか、重要なところが『よくあるご質問』とかに分散して隠されていたりするので若干の自業自得感もありそうです
* プロポーズ対策ですか、ミラコの部屋に関して、どの要素が好みにマッチしたのかにもよりますが…
    * ハババル：お気に入りだったなら直球で狙って良いでしょう、ただし園内との距離感、隣室との距離感を考慮するとタイミング次第では**ステージ上で公開プロポーズ状態**となりますので、好みによっては閉園後の夜景をバックにする等の考慮が必要かも知れません、あと時期次第では噂の2階増設部分（あくまでも噂）に当たりたくない／当たりたいも注意しておいた方が良いでしょう
    * ハバグラ：高層階が好みから外れるなら次点でしょうね、ただ屋根付きを取れると外から見たときのゴージャス感とプライベート感がありつつ雨に強いというメリットがありますね、部屋は狭いですから室内は避けたほうが無難でしょう
    * グラン棟：価格が跳ね上がりますが、プロポーズしたのはファンタジースプリングス開業の年にグラン棟で、というのは記憶に残りますよね、以下は現時点で判る範囲での推測ですが…
        * グランドルーム系：1泊40万オーバー、高層階なのでハババルのようなパークとの一体感に乏しいかも知れませんが恐らくミラコやランホ等TDR全体が見渡せるはず、最上位グレードで何しろ部屋が広い
        * テラス＆アルコーヴ：1泊35万前後、低層階に設定されているのでスプホの中ではハババルに近い位置づけかも知れません、テラスに出ても少し部屋側に寄ればハババルよりはプライベート感がありそうに見えます
        * 他の部屋：どの部屋もそれなりに良いとは思いますが、ハババル的な低層階保証がなくテラス系は恐らく高層階、ただアルコーヴ4-7辺りは予約が取りやすいというメリットはありますね
        * グラン棟は共通で『**部屋が広くて複数部屋あるので室内でも散らかっていない場所を維持しやすそう**』と『**全室バルコニー／テラスには屋根部分がありそう**』という雨や風に強い特徴がありそうですね、あとは**ラリベでミッキーにプロポーズ結果を報告する遊び**も出来そうです
        * まだ手すりやパーク内背景を画角に収める位置関係が不明なので、記念の写真や動画を残すなら三脚を持ち込んでも良いかも知れませんし、テーブルを併用する前提の机上三脚でも少し高めに調整出来る方が確実な気はします（画角を水平だけでなく見下ろしにも出来るように）
        * 全体が見渡せるのが良いということなら、グラン棟でもエントランス寄りの9階まである棟がアサインされる部屋が良いかも知れませんね
    * ファン棟：外に出てパークの空気を感じたいなら最難関のスプリングスサイドのバルコニーですが、難易度の割にパークとの距離感がありそうなので、ここは対象外でも良い気はします
    * いずれにしろ一般論でしかないので、最終的には**相手の好み次第**です、例えばハババルでプロポーズしたとして『パークのゲストに祝福されているようで嬉しい』と感じるか『パークから丸見えで晒し者みたいで恥ずかしい』と感じるか、グラン棟にしても『ゴージャスな体験で良かった』『ここでお金を使うなら別の機会（式、新婚旅行、引っ越し等）のために貯めて欲しかった』と人それぞれですからね
    * 色々な意味で**健闘を祈ります**
* 11時争奪戦は初手の読み込みやリロードが通るかは基本的に運次第なので、複数画面で挑んで安定性を上げたいところですね
    * 公式は待合室のしきい値を状況に応じて変更しているように見えるので、スプホブームがある程度は安定してきたので通常運転に戻す（待合室に入りにくくなる）ことで中が少し重くなったのかも知れませんね
