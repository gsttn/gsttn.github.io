# TDRホテル宿泊／レストラン予約 攻略情報

## 概要

TDRの希少価値が高く予約確保が困難なホテル宿泊とレストラン予約に役立つ情報を記載します。
具体的に言うと、宿泊ならミラコスタのテラスルーム・ハーバーグランドビュー／ハーバービュー、バルコニールーム・ハーバービュー、レストランならハイピリオン・ラウンジやドリーマーズ・ラウンジ辺りの予約確保を目標としている方に向けた情報です。

このサイトを見て何かを実践すれば必ずしも確実に予約が取れるということはありませんが、最低限度の知識や対策を把握することで争奪戦のスタートラインに立てる程度にはなれると考えております。
むしろ、このサイトの一番の存在意義は色々と情報に触れることで『自分も拾えるのではないか？』というモチベーションの維持、つまり**心が折れそうなときに自分を奮起させること**なのかも知れません。

## 最近の界隈の様子/中の人コメント

* ぼんやりと楽天のアフィリエイトリンクは置いていますが、ふと楽天トラベルの公開APIを叩いてディズニーリゾート周辺ホテルの空きを検索してみても良いかなぁ、と思うなど
    * 実際、レア枠（外に出られるミラコ部屋等）は出てこないと思うので、楽天経済圏にどっぷりな人向けですが
    * AIさんに色々と書いてもらいながら気がついてしまったのですが、シンプルに楽天トラベルで検索すれば良いだけの話という気も…
    * と思ったのですが、宿泊料金が公式サイトより概ね10%高いので、ポイント還元率が10%あればややお得みたいなレベルで割高感は否めないですね
    * 理屈上は同月の楽天市場における購入金額に楽天トラベルのSPU 1%分を計算に入れたりもできるので、割安に持って行くのも不可能ではないと思うのですがね
    * さらにミラコならまだハーバービューの部屋があるようですが、スプホはエントランスサイドとベイエリアサイドなので無理に監視するほどのこともないですよね
    * 実装着手の時点で既にモチベーションの97%くらいが消滅、お蔵入り不可避ですね…（苦笑
