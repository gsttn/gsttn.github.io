# TDRホテル宿泊／レストラン予約 攻略情報

TDRの希少価値が高く予約確保が困難なホテル宿泊とレストラン予約に役立つ情報を記載します。
具体的に言うと、宿泊ならミラコスタのテラスルーム・ハーバーグランドビュー／ハーバービュー、バルコニールーム・ハーバービュー、レストランならハイピリオン・ラウンジやドリーマーズ・ラウンジ辺りの予約確保を目標としている方に向けた情報です。

このサイトを見て何かを実践すれば必ずしも確実に予約が取れるということはありませんが、最低限度の知識や対策を把握することで争奪戦のスタートラインに立てる程度にはなれると考えております。
むしろ、このサイトの一番の存在意義は色々と情報に触れることで『自分も拾えるのではないか？』というモチベーションの維持、つまり**心が折れそうなときに自分を奮起させること**なのかも知れません。

## 最近の界隈の様子

* [ファンタジースプリングスエリアのバケパ](https://reserve.tokyodisneyresort.jp/news/detail/2464/)争奪戦が発表されましたね
    * 入場保証＋マジックチケット込み、6月はマジックチケットがあっても待ち行列エリアをゆっくりと眺めることが出来そうですね
    * あと、中の人はマジックチケットにおけるビジーバギーの扱いを勘違いしていたようですね、優先列が使えないだけでビジーバギーも扱えるのですね
    * 既報通りですが、スプホのグランシャトーは当然対象外
    * ファンタジーシャトーが設定されているプランは割りと多め
    * ただし『ホテルに泊まって優雅に過ごす』バケパはファンタジーシャトーが対象外
    * 一部バケパの予約対象期間は06/05からとなっていますが、ファンタジースプリングスホテルを選択可能なのは06/06以降でしょう、[06/06の運営内容](https://www.tokyodisneyresort.jp/hotel/topics/info/operation/fsh_operation.html)でスプホは12時オープンが明記されていますので
    * つまり、バケパでスプホ宿泊狙いの方は『バケパは先行して06/05から泊まれるってこと？』というリスクを取ったチャレンジは高負荷な状態ではオススメ出来ませんね
* [ファンタジースプリングスエリアのバケパ](https://reserve.tokyodisneyresort.jp/news/detail/2464/)争奪戦、なかなかの混乱でしたかね
    * 実は中の人は忙しくてあまり見ていない
    * これを書いている22時時点でも、まだ1時間以上の待合室が展開されている
    * スプホはほぼ全滅してエラー落ちを狙っている雰囲気
    * ファンタジースプリングスに入りたいだけなら、大人しくスプホ以外で取れば割りと空きがある
    * ちなみに01/16の22時すぎの時点でも、まだスプホの空きが落ちてくるケースを目撃しているで、このレベルの争奪戦ではどうやっても長丁場は不可避
        * 一応、22時半を過ぎた段階でも6月末の方であれば何度かスプホの空き落ちは確認
        * もっと言ってしまうと、22時半過ぎでも割りと普通にローズコートサイドを掴めたりしたので、根気強く狙っている人の幸運を祈るばかり…
        * あ、それから、今日のような待機時間が長い日は**待合室を抜けたら初手ログインが基本**で、すぐにバケパを確保しに行くとオプションを選択しているうちに待合室に追い出されてログイン画面で詰む（無限グルグル）ので要注意
    * さらにちなみに初日のファンタジースプリングスエリアに入るのが目的なら06/05泊（スプホ以外）のプランがまだ普通に残っているので、初日にマジックチケットで入ること自体は全く難しくないとも言える
    * つまり、多くの方の狙いはスプホ、02/08は史上最大規模の激戦となるかも

* ハババル4名部屋が3部屋になった理由までは不明ですね、バケパ枠からの移行か2階枠の関連か、効果は不明ですがリクエストは出すようにした方が無難かも知れないですね
