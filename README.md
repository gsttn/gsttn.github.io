# TDRホテル宿泊／レストラン予約 攻略情報

## 概要

TDRの希少価値が高く予約確保が困難なホテル宿泊とレストラン予約に役立つ情報を記載します。
具体的に言うと、宿泊ならミラコスタのテラスルーム・ハーバーグランドビュー／ハーバービュー、バルコニールーム・ハーバービュー、レストランならハイピリオン・ラウンジやドリーマーズ・ラウンジ辺りの予約確保を目標としている方に向けた情報です。

このサイトを見て何かを実践すれば必ずしも確実に予約が取れるということはありませんが、最低限度の知識や対策を把握することで争奪戦のスタートラインに立てる程度にはなれると考えております。
むしろ、このサイトの一番の存在意義は色々と情報に触れることで『自分も拾えるのではないか？』というモチベーションの維持、つまり**心が折れそうなときに自分を奮起させること**なのかも知れません。

## 最近の界隈の様子/中の人コメント

* 特にシーに居るとき、更に言うとケープコッドを通ると思うのですよね、**果たして次にダッフィの一味に加わるキャラの属性はどんなものなのか？**と…
    * 種族的には、熊→熊→猫→兎→犬→亀→狐、と来た訳で、まさかの爬虫類のカメが入ってきているので、哺乳類とも限らないのですよね
        * 哺乳類系なら、猿系、狸やアライグマ辺りを予想しますが、おそらく鼠はスターへのリスペクトと発端となる物語的なアレもあるので可能性は低いかな、と
        * ゾウとかキリンは特徴を捉えようとするとサイズ感や造形的に着ぐるみやぬいぐるみにしにくい気がしますね
        * あとは馬系や鹿系も着ぐるみで首を短くすると判りにくくなりそうですし、首を伸ばすとクリスマス系ショーのトナカイのような怖さが勝ってしまいそうで、これも難しいでしょうか
        * 狼やハイエナ、あとは虎、ヒョウ、ライオン等の犬系と猫系も既存の犬猫とビジュアル的に大きく区別しにくいのがデメリットですね
        * 哺乳類以外なら、爬虫類は既にカメが出ているので、鳥類から何か来ても良いかも知れませんね
        * 着ぐるみにすると翼を手にする不自然さはあるかも知れませんが、偉大なアヒルの先輩も居るのでここは大きな問題にならないでしょう
        * さすがにダックは外すとして、ホセ＆パンチートと敢えて同じにすることもないかな、と考えるとペンギンとかはグローバルに使いやすいかな、と
        * 極地の氷が融解して云々という環境意識みたいな方面にも見せやすいですし、2足歩行の着ぐるみへの親和性も高そうです
        * ただ、共通デザインっぽくなっている目とその周囲の部分、そこに付いているマズル（口や鼻先のある飛び出た部分）の存在もあるので、これらをデザインしやすい鳥の種類というとフクロウという可能性もありそうでしょうか（フクロウにマズルはなさそうですが）
    * 次に職業とか得意技ですが、少年→少女→絵描き→ダンサー→料理人（パティシエ）→ミュージシャン→探偵（？）、ですか
        * 流石に普通にビジネスマンとか官僚とかはなさそうですし、子どもに人気があっても世界観的にYouTuberとかプロゲーマーもなさそうでしょうかね
        * あまり既存メンバーの得意分野を侵食せず、それでいてイベント時の相乗効果が望めそうな得意分野はなかなか難しそうですね
        * 突然ステージ上でもないのに演技を始める役者、あるいは手品を始める手品師、やはりパーティにヒーラーは欲しいので医師や看護師の医療系、さすがに彫刻家とか歌手は猫と亀の分野を侵食しそうですかね
    * そしてキャラ立ちさせるために設定される色ですが、薄熊色→薄熊ピンク→薄緑→薄紫→黄→緑→ピンク、という感じですかね
        * 割とくすんだ感じである程度の大人でも日常使いしてもらいやすい色にするケースが多く、あまりパキッとした色使いはしないと思っていたのですが、カメの緑は割とパキッとしているかも知れませんね
        * とすると、パッと見で入っていないのは青系、ピンクではない赤系、赤を少し黄色に寄せてオレンジ系、という辺りが候補でしょうか
        * 案外、直球の白や黒、それをくすませてグレー系とかも日常使いはしやすい気もするので、なくはないのかな、と…
    * で、組み合わせとして何点か予想してみると…
        * サル系（チンパンジーやゴリラも含む）をベースに、器用さと賢さのイメージで手品師の属性、手品師なので黒系のグレーな子、地味でしょうかね
        * ペンギンをベースに、赤十字をイメージして赤を背負わせて医療系の特技を持った子、ヒーラーは表現が難しそうなので薬草や治療法の知識系な立ち位置ですかね
        * 残ったアライグマをベースに、何もないところで役に入り込む演技属性の役者、色は特に由来はないが残り物のくすみ青系の子、思いつきの寄せ集め感が…（苦笑
    * なんて妄想をするのも楽しみ方のひとつ、ということで、皆様はどんな予想あるいは妄想をしますかね？（微笑
        * あ、AIさんと相談しても面白いかも知れませんね

    * 精度や用途に関しては、現状では利用者自身のIT系な前提知識のレベルによって得られる結果レベルもかなり違いますし、意図しない挙動の結果が得られても動作が読めないと色々と困ったことになるリスクは大いにありますね
    * 例えば、スクリプト的な部分が全く判らないで『生成AIに作らせて実行してみる』というのは、『ここにある白い粉（小麦粉、ベーキングパウダー、片栗粉、塩、砂糖、重曹、クエン酸、洗濯用粉末洗剤、食器用粉末洗剤等を同じ場所に配置）でパンケーキを焼いて』と子供に作らせて、出来上がったモノを疑いもなく試食して確認する、みたいな状態なのですがね…（苦笑
    * つまり、**いわゆる自己責任的に実行した結果として、アカウント凍結やご指名で追放されても仕方ないという覚悟があるのか** という点は若干気になります
    * とは言え、**スクリプト作成に生成AIが活用される流れは変えようがない** と思いますし、誰かが『成功したスクリプト』を公開したらその時点で試合終了という点は以前から変わらないですね
        * 目をつけられて足を洗っただけの中の人が言うのもアレですが、学習と修行の成果で正解に辿り着いた人間が必ずしも道徳的であるという保証はないですから
        * もちろん確率的には誰もが生成AIで作れることで母数が増える現状の方が面白半分や自己顕示欲でスクリプト公開されるリスクは格段に高いですが…
    * とりあえず、**常に状況が変わっていくのは自然なことなので仕方ないですし、予約を取る利用者側の道徳やお行儀に期待するのは無理がある** でしょうから、**公式側で何等かの対策を考える必要に迫られている** と考えるのが妥当なところでしょうか
        * その結果、通常利用している人も簡単に不正検知に引っかかったり無用な待合室に放り込まれてサイトの使いにさに拍車がかかる、というのも割とあり得る話だと思いますが…
        * 例えばグランシャトーのラリベ予約方式の変遷を見ていると、利用者全員が等しく使いにくくなるなら公平である、みたいな思想を感じるのですよね
    * 公式さんが短期的にシステム的に対策するなら、待合室に戻しまくるとか諸々のTTLを減らす系でしょうかね、併せてペナルティ期間を延長すれば他の利用者への影響の軽減も可能でしょうか
        * とは言え、IPアドレスを変えてしまえばどうとでもなるので、万全の対策とは言い難いのですが、足止め効果はあるという程度でしょうか
        * もう少し手間をかけて意地悪をするなら、ペナルティページを出し分けしてAPIを叩いた子には通常の空き無し等と同じレスポンスを返してあげる等でしょうか、前段で返せば負荷の影響は無視できるレベルなので
    * なお、単純に興味があるという点では、どんなプロンプトを渡して、どんなスクリプトを実行したのかは見てみたい気もしますね
        * とは言え、そのネタでページを作ると、ダメ出し≒改善点の指摘になってしまうので、外には出せないネタということで中の人の個人的な楽しみにしかならないのですが…（苦笑
