# TDRホテル宿泊／レストラン予約 攻略情報

## 概要

TDRの希少価値が高く予約確保が困難なホテル宿泊とレストラン予約に役立つ情報を記載します。
具体的に言うと、宿泊ならミラコスタのテラスルーム・ハーバーグランドビュー／ハーバービュー、バルコニールーム・ハーバービュー、レストランならハイピリオン・ラウンジやドリーマーズ・ラウンジ辺りの予約確保を目標としている方に向けた情報です。

このサイトを見て何かを実践すれば必ずしも確実に予約が取れるということはありませんが、最低限度の知識や対策を把握することで争奪戦のスタートラインに立てる程度にはなれると考えております。
むしろ、このサイトの一番の存在意義は色々と情報に触れることで『自分も拾えるのではないか？』というモチベーションの維持、つまり**心が折れそうなときに自分を奮起させること**なのかも知れません。

        * グランドシャトー：33万～47万くらい
        * ディズニーランドホテル：スタンダード系で10万前後、ただしファミリー向けは部屋の広さで20万級まであり、キャラクター系は11～14万くらい、コンシェルジュ系は13～17万くらい

        * ミラコスタ：トスカーナ系とヴェネツィア系が12～14万くらい、ピアッツァ＆ハーバー側（パーシャルを除く）で14～16万くらい、バルコニー＆テラスは20～22万、それ以外のスペチで17～20万…マジスカ…

        * セレブレーションホテル：ほぼ4～6万くらい

        * ここまで価格差があるとハーバーショーが微妙な状況なら、ミラコのスペチ系よりランホのバルコニーから双眼鏡で城を眺める方がコスパ良いのでは、という声も出てきそうですね
        * さすがにランホとファン棟というデラックス系とトイホとの価格差は歴然としているけれど、アンバは『デラックス部屋もあるモデレート』とも言えそうな絶妙な価格帯ですね
        * あるいは逆にセレブレとトイホの差が小さい点を重視するなら、今となってはセレブレとトイホをまとめて両方バリュー級で良いのでは？と思えますね
        * 例えばデラックス級をベース部屋が10万辺りと定義するとランホとファン棟が該当、ちょっと良い部屋が14万辺りのホテルと定義するとランホとファン棟にアンバも該当、いずれにしろミラコは上のグレードとなりそうで、さすがデラックス四天王を率いるポジションですね
        * とは言え、ミラコとグランドシャトーには明確な価格差があるので、これをまとめてラグジュアリーというのは価格的にもサービス内容的にも現状では無理があるでしょう
        * 価格だけを見た体感としては『ラグジュアリー：グランドシャトー』『デラックス：ミラコ』『モデレート：アンバ＆ランホ＆ファンタジーシャトー』『バリュー：トイホ＆セレブレ』くらいの感覚になりそうですが、どうでしょうかね？
        * さておき、スプホ開業時によく見かけたグランドシャトーとミラコのテラス系の比較ですが、当時は2倍以上の価格差がある（宿泊日にもよる）状態でしたが、ほぼ1.5倍くらいになってきたのですね
        * ミラコのレア部屋における予約難度≒キャンセル拾いコストも考えたら、グランドシャトーの方がミラコのテラス＆バルコニーよりコスパに優れるという評価があっても良さそうな気がしますね
        * これも逆に言うと、そのキャンセル拾い稼働コストと宿泊価格アップを含めてもなお争奪戦をするくらいの体験価値がミラコのテラス＆バルコニーにはある、と考える人が多いということなのでしょう
        * ま、コスパで語ってはいけないという意味でも、グランドシャトーは別格ですが…
* 梅雨時といえばハロウィーン、真夏と言えばクリスマス、残暑厳しい年末年始、それがハイピジャンキーの正しい夏の過ごし方
